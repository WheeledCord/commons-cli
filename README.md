## Installation

### From PyPI (recommended)
```bash
pip install commons-cli
```

Then run:
```bash
commons-cli
# or
commons
```

## Development

### Local Setup
1. Install dependencies:
```bash
pip install -r requirements.txt
```

2. Run from source:
```bash
python main.py
```
